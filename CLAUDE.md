# Базовые правила разработки

## Язык коммуникации

- Все объяснения, комментарии к коду и ответы — на русском языке
- Все документы, создаваемые GSD-агентами (PROJECT.md, REQUIREMENTS.md, ROADMAP.md, codebase maps, research files и т.д.) — писать на русском языке

When adding issues to GitHub that are meant to be picked up by an AFK agent, use the `Sandcastle` label.
